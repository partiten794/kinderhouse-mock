使い方ガイド(保護者向け)

このアプリでできること

月ごとの利用申込み・変更・キャンセルと、キンダーハウスからのお知らせの受け取りができます。毎月の紙やフォームでのやり取りはこのアプリに一本化されます。

ホーム画面の上には 利用履歴 ・ 会員区分の変更 ・ 問い合わせ ・ 使い方ガイド の4つのメニューが並んでいます。新しい返信が届くと「問い合わせ」に●が付きます。

※ログイン・アカウント管理の機能は今後追加予定です。

毎月の申込みの流れ

  1. 画面を開くと、翌月のカレンダーに先月と同じ利用パターンが下書きとして入っています(左端が藤色の点線の日)。
  2. 変えたい日をタップして、お迎え時間を選びます。土曜日・学校休業日は入室時間も選びます。バス利用はスイッチでONに。
  3. 利用しない月や白紙から入れたいときは、リスト右上の「クリア」で下書きを消せます。
  4. 最後に画面下の「申込みを送信」を押して完了。送信前の概算料金も下に表示されます。

変更・キャンセル

  • 締切(前月5日)まで: 自由に変更できます。
  • 締切後〜利用前日の15時まで: 同じ画面からそのまま変更・追加・キャンセルできます。
  • それ以降(前日15時〜当日): アプリでは受け付けず、キンダーハウス携帯(080-XXXX-XXXX ※実際の番号はおたよりに記載)へお電話またはSMSでご連絡ください。該当日には「電話で」と表示されます。
学園を欠席する場合は学童にもご連絡ください(さくら連絡網でクラスに連絡済みの場合は不要)。
  • キャンセルは日をタップ →「この日の利用をキャンセルする」。
  • 当日のキャンセルやルールにないキャンセルは、料金がかかる場合があります。

利用履歴

ホーム画面上部のメニュー「利用履歴」から、月ごとの利用日数・合計時間・請求額を確認できます。月をタップすると、その月の日ごとの明細も見られます。申込み中の月は概算で表示されます。

会員区分の変更・退会

ホーム画面上部のメニュー「会員区分の変更」から、レギュラー・準レギュラー・スポットの変更と退会を申請できます。

  1. 変更後の区分を選びます(準レギュラーの場合は利用する曜日を3日選びます)。
  2. 適用を開始したい月と、理由・ご相談(任意)を入力します。
  3. 「この内容で申請する」を押すと申請完了。画面上部に「申請中」と表示されます。
  4. キンダーハウスが承認すると表示が「承認されました」に変わり、その月から料金計算も自動で切り替わります。
変更は希望月のひと月前までにご申請ください。お電話で伝えていただいた場合はスタッフが代理で登録し、同じ画面に反映されます。

同じ画面の下に、これまでの区分の履歴(いつからどの区分だったか)も表示されます。

問い合わせ

ホーム画面上部のメニュー「問い合わせ」から、キンダーハウスへメッセージで質問・相談ができます。返信も同じ画面に届きます。自分のメッセージには相手が読んだかどうか(既読・未読)が表示されます。

当日の変更・欠席などお急ぎの用件は、問い合わせではなくお電話でお願いします。

毎月のサイクル

  1. 受付開始前:「学園カレンダー設定」で翌月の休園日・行事(始業日・入学式・土曜行事など)を登録します。ここで設定した内容がそのまま保護者の申込み画面になります(「◯日は土曜日です」のような設問の手作りは不要)。
  2. 受付中(〜前月5日): 申込みは自動で集まります。締切後も前日15時までの変更はアプリ経由で反映されます。
  3. 利用当日:「日別利用者」で出席簿を確認。電話・SMSで受けた直前変更やキャンセルは、その児童の行を直接修正し、メモ欄に経緯(例: 電話連絡 8:40)を残します。
  4. 月末:「月次集計」で内容を確認し「この月を締める」。以降その月の数字は固定されます。
  5. 請求: 締めた集計から「CSV(銀行引落用)」を出力して引き落とし手続きへ。「利用明細」から全員分の明細PDFを一括出力して配布します。

日別利用者の見方

  • 確定 申込みどおりの利用予定
  • 昨日変更 24時間以内に変更があった行(メモ欄に変更内容)
  • 当日キャンセル 電話などで当日連絡があった行

「出席簿を印刷」で紙の出席簿、「CSV」で表計算用データを出力できます。

お迎え実績の転記

お迎えノートに記入された実際のお迎え時間は、同じ画面の「お迎え実績の転記」に入力します。初期値は予定時間なので、予定どおりなら「予定どおり」を1タップ、ズレていたら±5分ボタンで合わせるだけ。スポットの追加時間や延長保育の料金は自動で計算され、月次集計に反映されます(手計算・Excel転記は不要)。

予約を管理者が修正する(電話・窓口で受けた変更)

「日別利用者」の各行の右端「修正」から、その日の予定を管理者が直接書き換えられます。締切後でも前日15時を過ぎていても修正できます(保護者が操作できない時間帯の電話連絡はここで受けます)。

  1. 状態を選びます —「利用する」か「欠席」。欠席を選ぶと退室時間・バスの欄は消えます。
  2. 利用する場合はお迎え時間を選び、必要ならバス利用を切り替えます。
  3. メモに経緯を書きます(例: 母より電話 8:40。祖母がお迎え)。ここに書いた内容が出席簿のメモ欄に出ます。
  4. 受付経路(電話/窓口/メール/スタッフ判断)と対応スタッフを選んで「この内容で修正する」。

保存すると、修正内容が月次集計・請求・保護者の画面にそのまま反映されます。同時に「管理者による修正の記録」に、いつ・誰が・何を・どの経路で受けて変えたかが自動で残ります。同じ記録は利用者マスタの各児童のページにも表示されます。

状態バッジ 管理者修正 は、保護者の申込みではなくスタッフが書き換えた行であることを示します。

メモ・申し送りの残し方

  • その日のこと(欠席の連絡・お迎えの人が違うなど)→「日別利用者」の修正のメモ欄。出席簿に出ます。
  • ずっと気をつけること(アレルギー・配慮事項・バスの降車場所など)→ 利用者マスタの備考。全スタッフが見る前提の欄です。
  • 経緯として残したいこと(相談の内容・伝えたこと)→ 備考の下の日付つきメモ、電話で受けたものは電話・口頭の記録へ。日付と入力したスタッフ名が自動で付きます。

会員区分の変更

  • 保護者がアプリから申請した場合: 利用者マスタの上部「会員区分の変更申請」に並びます。タブに未処理の件数バッジが出ます。内容を確認して「承認」を押すと、その月から区分と料金計算が切り替わり、保護者にも通知されます。相談が必要なら「保留」にして問い合わせでやり取りします。
  • 電話で受けた場合: 一覧から児童を開いて「区分を変更」。変更後の区分・利用曜日・適用開始月・受付経路と理由を入れて登録します(代理登録)。
  • どちらの場合も変更の記録が残り、適用開始月より前の確定済みの請求には影響しません。

料金の計算ルール

レギュラー/準レギュラーは定額(月〜金)、土曜・休業日の利用はスポット(300円/30分)として自動加算します。延長(700円+おやつ300円 ※運用確認中)も自動計算。8月・3月のレギュラー/準レギュラー増額も自動です。

※料金改定時は料金マスタの更新だけで全体に反映されます(実装時)。

問い合わせ対応

「問い合わせ」タブに家庭ごとのスレッドが並びます。未読があるスレッドには件数バッジが付き、開くと既読になります(保護者側にも既読が伝わります)。返信は入力して送信するだけ。自分の返信を保護者が読んだかどうかも吹き出しの下に表示されます。
